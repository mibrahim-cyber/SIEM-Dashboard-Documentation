﻿# Residual risks

This is a lab project, and it's honest about what it doesn't solve. It runs on SQLite and a single Node process, so it isn't built for high availability or multi-tenant isolation. The threat feeds are rate-limited free tiers, the GeoLite2 data is approximate, and the demo users exist for teaching and must be rotated before any real exposure. Listing the gaps is part of the security story, not something to hide.
