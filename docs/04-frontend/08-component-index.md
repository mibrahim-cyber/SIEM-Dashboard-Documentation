﻿# UI module index

Index linking each UI module doc to its React source file under `src/components/`.

This page maps each UI module doc to the component that implements it under `src/components/`. The dashboard imports around thirty of them, one per module, and the names line up with the menu, so `AlertManager.jsx` backs the Alert Manager view and so on. Use it as a jumping-off point when you want to get from a screenshot to the code behind it.
