# Logic gates overview

The logic-gate pages cover how a rule's individual conditions combine into a decision. A rule isn't always a single test; it can be several conditions joined with boolean logic, sometimes counted over a time window. These pages walk through how those conditions compose, how stateful rules differ from stateless ones, and how to tune away false positives. Think of them as the theory behind the per-rule checks.
