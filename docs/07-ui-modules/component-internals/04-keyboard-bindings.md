﻿# Keyboard bindings

HABIBI-SIEM UI modules share implementation patterns documented here for coursework extensions.

## Purpose

Global shortcuts open command palette, focus search fields, and navigate major sections where enabled. Bindings are suppressed when typing in text areas to avoid hijacking ingest paste.

## Runtime behavior

Document shortcuts in appendix keyboard reference. Conflicts with browser defaults (Ctrl+K) may require OS-specific notes for macOS vs Windows labs.

## Operator and developer notes

Students using screen readers may need mouse-equivalent routes documented in lab handouts when shortcuts are optional extras.
