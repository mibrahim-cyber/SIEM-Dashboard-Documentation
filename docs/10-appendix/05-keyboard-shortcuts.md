﻿# Keyboard shortcuts

The main keyboard entry point is the command palette, opened with Ctrl+K (or Cmd+K on a Mac). From there you type to jump straight to any module instead of reaching for the sidebar, which is the quickest way to move around during a demo. Shortcuts don't fire while you're typing in an input or textarea, so the palette won't fight with form fields.
