﻿# Lab exercises

The lab exercises are short, hands-on tasks that walk through the app the way an analyst would: run a simulated campaign and watch the alerts correlate, trace a single source IP across the views, or trip a rule on purpose and then tune it back down. They're meant to make the detection and triage flow concrete instead of just described. Each one lines up with a module the docs already cover.
