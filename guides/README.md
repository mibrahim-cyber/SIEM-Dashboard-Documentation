# Module guides

Screenshot walkthroughs and **per-topic detailed pages** for each sidebar section in the SIEM Dashboard.

[![Guides](https://img.shields.io/badge/modules-27-f97316?style=flat-square)](./)
[![Pages](https://img.shields.io/badge/topic_pages-190-a78bfa?style=flat-square)](./)

</div>

Each module folder contains:

| File | Purpose |
|:-----|:--------|
| [`01-how-to-use.md`](monitor/overview/01-how-to-use.md) | Quick steps + one screenshot |
| [`INDEX.md`](monitor/overview/INDEX.md) | Table of contents for all pages in that module |
| `02-*.md` … | **One standalone page per sub-topic** (800+ words each) |

Example, **Configure → Correlation Builder:**

- [INDEX.md](configure/correlation-builder/INDEX.md) lists all 7 pages
- [02-what-correlation-means.md](configure/correlation-builder/02-what-correlation-means.md)
- [03-correlation-builder-ui.md](configure/correlation-builder/03-correlation-builder-ui.md)
- … through `08-real-world-examples.md`
